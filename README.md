# bs-action-sync

一个跨终端交互同步的玩具。


## 快速开始

### 安装

```bash
# 全局
yarn global add @bszhct/bs-action-sync

# 局部
yarn add @bszhct/bs-action-sync
```

### 使用

```bash
bs-action-sync start [--address] [--port]

# bs-action-sync start -a http://127.0.0.1:9001
# bs-action-sync start -a http://127.0.0.1:9001 -p 9002
```

### 查看帮助

```bash
bs-action-sync -h
```
